# レポートアーカイブ

このディレクトリには、自動生成されたHTMLレポートが保存されています。

## フォルダ構成

```
report/
├── daily/           # 日次レポート
│   └── daily_report_YYYY_MM_DD.html
└── weekly/          # 週次レポート
    └── weekly_action_report_YYYY_MM_DD.html
```

## 最新レポート

### 日次レポート
日次レポートは `daily/` フォルダに保存されます。
- ポートフォリオの現在値と損益状況
- AI/IT関連の重要ニュース分析
- 投資への示唆

### 週次アクションレポート
週次レポートは `weekly/` フォルダに保存されます。
- ポートフォリオの詳細評価
- 今週の推奨アクション
- 中長期的な投資戦略

## レポートの閲覧方法

1. **GitHub上で直接閲覧**
   - `daily/` または `weekly/` フォルダから目的のレポートを選択
   - GitHubの「Raw」ボタンからダウンロード
   - ダウンロードしたHTMLファイルをブラウザで開く

2. **GitHub Pagesで閲覧（オプション）**
   - リポジトリ設定でGitHub Pagesを有効化すると、直接ブラウザで閲覧可能

3. **GitHub Actionsのアーティファクトからダウンロード**
   - Actions タブ → 実行されたワークフロー → Artifacts
   - PDF版もダウンロード可能

## 更新スケジュール

- **日次レポート**: 毎日 JST 6:00 (UTC 21:00)
- **週次レポート**: 毎週金曜日 JST 20:00 (UTC 11:00)

## 注意事項

- レポートは日付ごとにファイル名が付けられ、履歴が保持されます
- 過去のレポートもこのディレクトリから参照可能
- GitHub Actionsのアーティファクトでの保持期間：
  - 日次レポート: 30日間
  - 週次レポート: 90日間
